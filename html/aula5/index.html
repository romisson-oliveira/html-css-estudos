Formatação de Texto - Html Semântico

Negrito e Itálico

Antigamente, com html 4, usava-se as tabs <b> e <i> para esta tarefa, mas elas não possuem significado e focam apenas na forma, sendo assim, são pouco semânticos.
Em html 5, usamos <strong> para negrito e <em> para itálico, pois este último serve para dar ênfase.

Essas tags não só mudam a forma das palavras mas também possuem sentido. Porém, para nós o que importa mesmo utilizando essas tags está em seu significado. Pois a forma como ele serão apresentados dependerá de como estruturamos no CSS.

Texto deletado

Em html 5 envolva o texto que deseja criar um risco deletado com a tag del → <del>

Texto inserido

O texto inserido é o exato oposto do texto deletado que vimos anteriormente. Neste caso, se colocarmos um texto qualquer entro de <ins> e </ins>, estamos dizendo que o texto está ali deve ser lido e você deve prestar atenção nele. É o que fazíamos anteriormente ao utliziar a tag <u> para sublinhar um texto.

Textos sobrescrito e subscrito

Use tag <sob> para usar número como expoente: x=52+5.
Use a tag <sub> para usar números assim: Ca2.


Trechos de código

Se em algum momento desejar inserir um trecho de um código de programação, usamos a tag <code> para dar um valor semântico ao trecho dentro da tag pra que HTML possa delimitar seu código. Inclusive, o trecho terá um efeito visual, fazendo com que as letras ficam no modo mono-espaçadas (monospace), o que facilita bastante a leitura do código. Observe:

Exemplo de código em Python
num = int(input('Digite um número: ')) if num % 2 == 0: print(f'O número {num} é par') else print(f'O número {num} é ímpar') print('Fim do Programa')

Observe que no código acima foi escrito com quebra de linha e identação no html, porém não foi respeitado no momento da exibição no navegador. Para resolver isso, colocamos todo o bloco que está na tag <code> dentro da tag <pre>. Esta tag será a tag pai da tag code, fazendo com que a tag filho seja exibida da exata forma em que está sendo escrita no código html. Observe:

"Citações"

Para usar um trecho use a tag <q> (do inglês quote, que significa citar). O texto que estiver entre <q> e </q> já vai receber automaticamente as aspas, mas não terá nenhum deslocamento. Essa técnica é mais usada quando queremos um citação no meio do parágrafo.

Para citar em bloco, usamos a tag <blockquote>, observe:

Por exemplo: quando questionado sobre o que pessoas inteligentes conversam, o humorista renascentista italiano, escritor e grastrônomo, Bartolomeo Sacchi, nascido em 1421 em Piadena, respondeu prontamente e com toda segurança a frase: de litteris e de armis, praestantibusque ingeniis. A frase está em latim e significa, em tradução livre, de letras, de armas e de gênios formidáveis. Uma forma de atribuir os conceitos que realmente deveriam estar relacionados aos contextos de suas conversas.

Abreviações

Em HTML5 usamos a <abbr> (abbrevition, em inglês)para definir que aquele texto dentro da tag representa uma abreviação de algum termo. O título da abreviação deverá representar o que aquela abreviação significa. Usamos o atributo "title" para isso e o valor dentro do atribulo será o que repesenta. 

<p>Estou estudando <abbr title="Hyper Text Markup Language">HTML</abbr> para criar sites.</p>

Estou estudando HTML para criar sites.

Quando você passar o mouse por cima da abreviação, verá o valor que está inserido no atributo title acima.
