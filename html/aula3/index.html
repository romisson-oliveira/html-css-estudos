Inserindo favicon e imagens em sua página html

Favicon

Favicon são aqueles ícones que ficam no topo da página, na aba do navegador.

Use sites como Favicon.io para escolher um favicon. Cuidado com os direitos autoriais.Algumas imagens possuem direito atrelados aos seus criadores ou detentores legais.

Não use imagem sem autorização.

Use a tag link:favicon para configurar a favicon. Atributo href deverá ser direcionado para o arquivo png ou ico.

Imagens

Para inserir imagem usamos a tag <img src=""> para adicionar uma imagem. No atributo search - src, insira o url ou o caminho para acessar a partir da pasta do projeto.
Atributo alt -> Serve para inserir um texto alternativo na imagem para ajudar na acessibilidade e na orientação do que se trata a imagem inserida.
